source
######

BL (botlib)
===========

.. autosummary::
    :toctree: code
    :template: module.rst

    bl				- bot library.
    bl.all			- contains all sub modules
    bl.bot			- bot base class.
    bl.clk			- clock functions.
    bl.csl			- console.
    bl.dbs			- database.
    bl.err			- errors.
    bl.flt			- list of bots.
    bl.hdl			- handler.
    bl.ldr			- module loader.
    bl.log			- logging.
    bl.shl			- shell.
    bl.tms			- time related.
    bl.trm			- terminal code.
    bl.thr			- threads.
    bl.trc			- trace.
    bl.typ			- typing.
    bl.usr			- user management.
    bl.utl			- utilities.

BOTD (channel daemon)
=====================

.. autosummary::
    :toctree: code
    :template: module.rst

    botd.cfg 			- edit configuration.
    botd.cmd			- basic commands
    botd.ent			- log and todo commands.
    botd.irc			- IRC bot.
    botd.rss			- feed fetcher.
    botd.shw			- show status.
    botd.udp			- UDP packet to IRC channel relay.
